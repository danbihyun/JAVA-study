# 03. 연산자

## 03-1. 연산자와 연산식 (p.129)

### 1. 연산자와 연산식

1-1.  
**연산자**는 피연산자의 수에 따라 단항, 이항, 삼항 연산자로 구분된다.

1-2.  
비교 연산자와 논리 연산자의 산출 타입은 논리 타입(boolean)이다.

1-3.  
연산식은 반드시 하나의 값을 산출한다. 연산자 수가 아무리 많아도 2개 이상의 값을 산출하는 연산식은 없다.

1-4.  
(1-3의 내용과 이어짐) 그렇기 때문에, 하나의 값이 올 수 있는 곳이면 어디든지 값 대신에 연산식을 사용할 수 있다.

### 2. 연산의 방향과 연산 우선순위

2-1.  
산술 연산(+, -, \*, /)은 연산 방향이 왼쪽에서 오른쪽으로 수행된다.

2-2.  
증감 연산(++, --)과 대입 연산(=, +=)은 연산 방향이 오른쪽에서 왼쪽으로 수행된다.

2-3.  
연산 우선순위가 정해져 있지만, 괄호로 감싼 연산이 최우선순위를 갖기 때문에 우선 순위와 관계 없이 먼저 연산을 하고 싶다면 괄호로 묶어준다.

2-4.  
논리 연산(&&, ||)은 산술 연산(+, -, \*, /)보다 우선순위가 낮다.

## 03-2. 연산자의 종류 (p.149 ~ 153)

### 1. 컴파일 에러 발생하는 이유

```java
byte b = 5;
b = -b;
int result = 10 / b;

System.out.println(result);
```

▼

```java
b = -b;
에서 컴파일 에러 발생
부호 연산을 하며 int 타입으로 바뀌기 때문
```

### 2. 출력 결과

```java
int x = 10;
int y = 20;
int z = (++x) + (y--);

System.out.println(z);
```

▼

```java
int z = (++x) + (y--);
++x: 값을 1 올린 후 x에 변수 저장
y--: 연산 후 y값 -1 이므로

int z = 11 + 20

System.out.println(z); 의 값은 31
```

```java
stop 의 값이 false 일 때 while 문이 돌도록 지정해주면 된다.
```

### 5. 잘못된 부분 발견하기

```java
int var1 = 5;
int var2 = 2;
double var3 = var1 / var2;
int var4 = (int)(var3 * var2);

System.out.println(var4);

// 5가 출력될 것이라고 예상했으나 4가 출력됨
```

▼

```java
double var3 = var1 / var2; // 2.5
int var4 = (int)(var3 * var2); // int로 강제형변환 됐으므로 2 x 2 = 4
```

### 7. 잘못된 부분 발견하기

```java
float var1 = 10f;
float var2 = var1 / 100;

if(var2 == 0.1) {

    System.out.println("10% 입니다.");

} else {

    System.out.println("10%가 아닙니다.");

}

// 10% 입니다. 가 출력될 것이라고 예상했으나 10%가 아닙니다. 가 출력됨
```

▼

```java
float var2 = var1 / 100;
의 값은 0.1f 이지만, float 타입의 0.1은 완전히 0.1이 아닌 숫자.
if(var2 == 0.1)
에서 0.1f == 0.1 을 연산하면 float == double 이므로 double로 승격되면서 0.1f가 미세하게 달라짐. (0.10000000149 이런 식으로)
그러므로 else 중괄호의 값인 10%가 아닙니다. 가 출력됨.
```

### 9. 코드 작성하기 (Exam090302.java)

### 12. 출력 결과

```java

```

▼

```java

false
```

### 13. 실행문 변경 (산술 연산자 삭제)

```java
int value = 0;

value = value + 10; // 10
value = value - 10; // -10

value = value / 10; // 0
```

▼

```java

value -= 10; // -10
value *= 10; // 0
value /= 10; // 0
```
